== Using inverse functions To Find Angle
[stem]
++++
\tan\theta = \frac{20.8}{7}
++++

[stem]
++++
\tan^-1\theta = tan^-1(\frac{20.8}{7})
++++

[stem]
++++
\theta = 71.5^\circ
++++
