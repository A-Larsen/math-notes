== Fromula
[stem]
++++
\lim_{h \to 0} \frac{f(x+h) - f(x)}{h} = f'(x)
++++

== Basic Rules

Constant Rule::
[stem]
++++
\frac{d}{dx} [c] = 0
++++

Power Rule::
[stem]
++++
\frac{d}{dx} [x^n] = nx^(n-1)
++++

Product Rule::
[stem]
++++
\frac{d}{dx} [f(x)g(x)] = f'(x)g(x) + f(x)g'(x)
++++

Quotient Rule::
[stem]
++++
\frac{d}{dx} [\frac{f(x)}{g(x)}] = \frac{f'(x)g(x) - f(x)g'(x)}{(g(x))^2}
++++

Chain Rule::
[stem]
++++
\frac{d}{dx} [f(g(x))]  = f'(g(x)) * g'(x)
++++
