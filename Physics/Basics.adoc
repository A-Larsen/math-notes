== Instentaneos Rate of Change
[stem]
++++
\lim_{\Delta \to 0} \frac{f(x + \Deltax) - f(c)}{\Deltax} = f'(x)\
or \
\lim_{x \to c} \frac{f(x) - f(c)}{x - c} = f'(c)
++++

=== Terms

[stem]
++++
f'(a) = v(t) -> \text{velocity}\
f''(a) = a(t) -> \text{acceleration}\
|f'(a)| = s(t) -> \text{speed}\
f'''(t) -> jerk
++++
